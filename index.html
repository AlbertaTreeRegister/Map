Welcome to the Alberta Tree Register Map

Select a tree on the map for more information or use the menu to:

1) Search for a tree by its name, address, or neighbourhood.

2) View a list of the top trees. Select a tree to navigate to its location.

3) View the photo gallery. Select an image to view it in fullscreen (desktop only) or select the tree name for more information.

4) Add a tree to the register.

5) Reset the map to its default position.
